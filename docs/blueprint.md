# **App Name**: Pasqua AI Chat

## Core Features:

- Chat Interface: A simple and clean chat interface to interact with the Gemini API.
- Gemini Integration: Send user prompts to the Gemini API with the configured API key.
- Response Display: Display the API response in a structured format.
- API Key Storage: Allow users to enter and save their Gemini API key.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to convey tranquility and focus.
- Background color: Light gray (#F5F5F5) for a clean, unobtrusive backdrop.
- Accent color: Dusty Violet (#D8BFD8) for subtle highlights and interactive elements.
- Clean, sans-serif font for optimal readability in chat bubbles and interface elements.
- Simple and intuitive layout with clear distinction between user input and AI responses.
